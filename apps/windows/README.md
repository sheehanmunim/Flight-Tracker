# Windows Apps

This folder contains the Windows-side app entrypoints and source code.

- `FlightTracker/`: desktop launcher source
- `DashboardHost/`: browser-host source
- `Shared/`: shared feeder/runtime code
- `Run-FlightTracker-Windows.cmd`: development launcher
- `Run-FlightTracker-Browser.cmd`: development browser-host launcher

If you just want to use the packaged app, go to `dist/windows/`.
