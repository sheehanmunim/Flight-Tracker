Chrome Direct Lab

This is the experimental browser-only track for direct USB access in Chromium-based browsers. It can request a USB device and inspect basic descriptors, but the full SDR demodulation and ADS-B decode pipeline is not implemented in the browser yet.

Use the Mac or Windows host mode for the working receiver path today. Use this page to explore Chrome-only WebUSB access and to stage a future direct-browser receiver workflow.
