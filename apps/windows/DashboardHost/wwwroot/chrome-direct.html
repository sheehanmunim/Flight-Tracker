Browser Receiver

This page can talk to an RTL-SDR directly from Chromium over WebUSB, decode ADS-B in the browser, and show live aircraft without using the Windows or macOS host decoder.

Browser-only receiver requirements: desktop Chromium or Chrome, a secure context such as https:// or http://localhost, and an RTL2832U-based USB dongle. Safari and Firefox do not expose WebUSB for this workflow.
